Extras Panel — UI Options

Choose where to place the extras quantity controls. Both options shown below using app styling.

Option A
Extras panel on the Order Summary page
A collapsible "Extras" section appears below the BOM table on the Order Summary page. Users add extras in context with the full BOM, and the right-side cost panel updates to reflect both BOM + extras totals.

Option B
Tabbed view — As Configured / Extras
Same Order Summary page, but the two lists are separated into tabs rather than stacked sections. The right panel always shows both subtotals and the full-order total regardless of which tab is active.
